# README.md

## Docker

YuriaCompose直下での実行の場合

```bash
# イメージのビルド
docker build ./botapp --tag botapp:latest

# イメージの実行
docker run botapp:latest
```

## デフォルトの記載

```
npm install
npm run dev
```

```
open http://localhost:3000
```

## このプロジェクトの立ち上げ

```bash
# honoプロジェクトの立ち上げ
npm create hono@latest

# nodejsを選択
```
